Reactмқҳ memoizing кё°лҠҘл“Ө (2021-04-14 мҲҳ)

м„ұлҠҘн–ҘмғҒмқ„ мң„н•ң memoizing кё°лҠҘл“Ө

React.memo()

ReactлҠ” componentлҘј renderingн•ң л’Ө мқҙм „ renderingкіј л№„көҗн•ҳм—¬ DOMмқҳ м—…лҚ°мқҙнҠёлҘј кІ°м •н•ңлӢӨ. л§Ңмқј componentлҘј React.memo()лЎң лһҳн•‘н•ңлӢӨл©ҙ, ReactлҠ” н•ҙлӢ№ componentлҘј renderingн•ҳкі , к·ё кІ°кіјлҘј memoizingн•ңлӢӨ. к·ёлҰ¬кі  propsмқҳ ліҖнҷ”к°Җ мқјм–ҙлӮҳкё° м „к№Ңм§Җ memoizingлҗң лӮҙмҡ©мқ„ мһ¬мӮ¬мҡ©н•ңлӢӨ.

мқҙл•Ң мЈјмқҳн•  м җмқҖ `propsм—җ лҢҖн•ң л¶Җл¶„л§Ң н•ҙлӢ№лҗ  лҝҗ stateм—җ лҢҖн•ң ліҖнҷ”лҠ”

мқҙлҠ” classнҳ•мқҳ componentLifeCycleмӨ‘ shouldComponentUpdateм—җ лҢҖмқ‘лҗҳлҠ” кё°лҠҘмңјлЎң, returnлҗҳлҠ” к°’мқҙ trueмқҙл©ҙ re-renderingмқҙ мқҙлЈЁм–ҙ м§„лӢӨ.

React.memoмқҳ л№„көҗл°©мӢқ custom

кё°ліём ҒмңјлЎң React.memo()лҠ” shallow copyлҘј нҶөн•ҙ propsлҘј л№„көҗн•ҳлҠ” кІғмқҙ кё°ліёлҸҷмһ‘мқҙлӢӨ. л§Ңмқј customн•ң л№„көҗлҘј мӣҗн•ңлӢӨл©ҙ, л‘җ лІҲм§ё мқёмһҗм—җ customн•ң л№„көҗ н•ЁмҲҳлҘј м ңкіөн•ҳлҠ” л°©мӢқмқ„ мӮ¬мҡ©н•ңлӢӨ.

мң„ мҪ”л“ңмҷҖ к°ҷмқҙ м—¬лҹ¬л°©мӢқмқҳ customн•ң л№„көҗл°©мӢқмқ„ нҶөн•ҙ л№„көҗлҘј н•  мҲҳ мһҲлӢӨ. мқҙл•Ң кё°м–өн•ҙм•јн•  м җмқҖ trueлҘј л°ҳнҷҳн•ҳл©ҙ re-renderingмқҙлқјлҠ” м җмқҙлӢӨ.

React.memoмқҳ мӮ¬мҡ©мқҙ нҡЁмңЁм Ғмқё мғҒнҷ©

React.memoлҘј мӮ¬мҡ©н•ҳлҠ” кІғмқҙ л°ҳл“ңмӢң м„ұлҠҘмқ„ нҡЁмңЁм ҒмңјлЎң л§Ңл“Өм–ҙмЈјм§ҖлҠ” м•ҠлҠ”лӢӨ.

  • renderingмӢңм—җ propsк°Җ лӢӨлҘёкІҪмҡ°
  • classнҳ• componentлҘј мӮ¬мҡ©н•ҳлҠ” кІҪмҡ° (shouldComponentUpdateлҘј мӮ¬мҡ©)

мҳҲлЎң мң„ л‘җ мғҒнҷ©мқҳ кІҪмҡ°м—җлҠ” propsк°Җ лӢ¬лқј memoizingмқҳ мқҳлҜёк°Җ м—ҶлҠ” л“ұмқҳ мғҒнҷ©мңјлЎң м“ёлӘЁм—ҶлҠ” props л№„көҗк°Җ мқҙлЈЁм–ҙм§„лӢӨ.

л”°лқјм„ң React.memoлҠ” к°ҷмқҖ propsлҘј мқҙмҡ©н•ҙ мһҗмЈј renderingлҗ  кІғмңјлЎң мҳҲмғҒлҗҳлҠ” л¶Җл¶„м—җ мӮ¬мҡ©н•ҳлҠ” кІғмқҙ к°ҖмһҘ мўӢмқҖ м„ұлҠҘмқ„ л°ңнңҳн•ңлӢӨ. (мҳҲлЎң кІҢмӢңнҢҗм—җ кІҢмӢңкёҖ 1000к°ң мӨ‘ нҠ№м • 1к°ңлҘј мҲҳм •н–Ҳмқ„ кІҪмҡ°, лӮҳлЁём§Җ 999к°ңлҠ” re-renderingлҗ  н•„мҡ”к°Җ м—ҶкІҢлҒ”)

мқҙлҘј м • нҢҗлӢЁн•ҳкё° м–ҙл өлӢӨл©ҙ react dev-toolмқҳ profilerлҘј мқҙмҡ©н•ҙ л№„көҗн•ҙліҙлҠ” кІғмқҙ лӮҳмқ„ л“ҜвҖҰ

useCallback unnecessary dependency

useCallback м—җм„ң лӮҳнғҖлӮҳлҠ” мҳӨлҘҳ мӨ‘ unnecessary dependency кІҪкі к°Җ мҪҳмҶ”м—җ лңЁл©ҙ, second parameterм—җ м“ёлӘЁм—ҶлҠ” к°’мқҙ л“Өм–ҙк°ҖмһҲлӢӨлҠ” кІғмқҙлӢӨ.

Line 9:5: React Hook useCallback has an unnecessary dependency: 'state'.

лӮҳмқҳ кІҪмҡ° кі„мҶҚн•ҙм„ң мқҙлҹ° кІҪкі к°Җ л°ңмғқлҗҳм—ҲлҠ”лҚ°, м•Ңкі ліҙлӢҲ н•ЁмҲҳ м•Ҳм—җм„ң мӮ¬мҡ©н•ҳлҠ” state лҳҗлҠ” propsлҘј лӮҙк°Җ мһҳлӘ» мқҙн•ҙн–Ҳкё° л•Ңл¬ёмқҙм—ҲлӢӨ. мқҙ м–ҙнңҳлҠ” л§җ к·ёлҢҖлЎң useCallback м•Ҳм—җм„ң мӮ¬мҡ©н•ҳлҠ” кІғмқ„ мқҳлҜён•ңлӢӨ.
