useRef

useRefлҘј мӮ¬мҡ©н•ҳлҠ” кІҪмҡ°лҠ” л‘җ к°Җм§ҖлЎң лӮҳлүңлӢӨ.

1) нҠ№м • DOMмқ„ м„ нғқн•  л•Ң

2) component лӮҙл¶Җм—җм„ң мЎ°нҡҢ л°Ҹ мҲҳм •н•  мҲҳ мһҲлҠ” ліҖмҲҳ кҙҖлҰ¬

мқјл°ҳм ҒмңјлЎң stateлҠ” 4лӢЁкі„лҘј л”°лҘёлӢӨ.

  1. state м„Өм •
  2. setStateн•ҳлҠ” н•ЁмҲҳ мӢӨн–ү
  3. stateліҖнҷ”м—җ л”°лҘё re-rendering
  4. ліҖкІҪлҗң state мЎ°нҡҢ к°ҖлҠҘ

мқҙмІҳлҹј мқјл°ҳм Ғмқё stateлҠ” re-renderingмқҙ мқҙлЈЁм–ҙм§„ мқҙнӣ„м—җм•ј ліҖкІҪлҗң stateлҘј мЎ°нҡҢн•  мҲҳ мһҲлӢӨ. н•ҳм§Җл§Ң, useRefлҘј мқҙмҡ©н•ҙ кҙҖлҰ¬н•ҳл©ҙ re-rendering м—Ҷмқҙ л°”лЎң мЎ°нҡҢк°Җ к°ҖлҠҘн•ҳлӢӨ.

л”°лқјм„ң лӢӨмқҢмқҳ кІҪмҡ°м—җ useRefлҘј мӮ¬мҡ©н•ңлӢӨ.

  1. setTimeout, setIntervalмқ„ нҶөн•ҙ л§Ңл“Өм–ҙм§„ id
  2. мҷёл¶Җ libraryлҘј мӮ¬мҡ©н•ҙ л§Ңл“Өм–ҙм§„ Instance
  3. scrollмқҳ мң„м№ҳ

useEffect

  • useEffectмқҳ мІ« лІҲм§ё parameterлҠ” н•ЁмҲҳк°Җ, л‘җ лІҲм§ё parameterлҠ” мқҳмЎҙк°’мқҙ л“Өм–ҙмһҲлҠ” л°°м—ҙмқ„ л„ЈлҠ”лӢӨ.
  • useEffectм—җм„ңлҠ” н•ЁмҲҳлҘј returnн•  мҲҳ мһҲлҠ”лҚ°, мқҙлҘј cleanup functionмқҙлқјкі  л¶ҖлҘёлӢӨ. мқҙлҠ” componentк°Җ м–ёл§ҲмҡҙнҠёлҗ  л•Ң нҳём¶ңлҗҳлҠ” н•ЁмҲҳмқҙлӢӨ.

л”°лқјм„ң л‘җ лІҲм§ё parameterлҘј м–ҙл–»кІҢ мӮ¬мҡ©н•ҳлҠҗлғҗм—җ л”°лқј нҳём¶ңл°©мӢқмқҙ кІ°мЎҚлҗңлӢӨ.

  1. л°°м—ҙмқҙ л№Ҳ кІҪмҡ°: componentк°Җ мІҳмқҢ лӮҳнғҖлӮ л•Ңл§Ң useEffectм—җ л“ұлЎқлҗң н•ЁмҲҳк°Җ нҳём¶ңлҗңлӢӨ.
  2. л°°м—ҙм—җ нҠ№м • к°’мқ„ л„ЈмқҖ кІҪмҡ°
    1. componentк°Җ мІҳмқҢ лӮҳнғҖлӮ  л•Ң нҳём¶ң
    2. н•ҙлӢ№ к°’мқҙ л°”лҖ” л•Ң нҳём¶ң
    3. н•ҙлӢ№ к°’мқҙ л°”лҖҢкё° м§Ғм „м—җ нҳём¶ң
    4. м–ёл§ҲмҡҙнҠёмӢң cleanup function нҳём¶ң
  3. л°°м—ҙ мһҗмІҙк°Җ мғқлһөлҗң кІҪмҡ°
    1. componentк°Җ re-renderingлҗ  л•Ңл§ҲлӢӨ нҳём¶ң
    2. м–ёл§ҲмҡҙнҠёмӢң cleanup function нҳём¶ң

useEffectм•Ҳм—җм„ң мӮ¬мҡ©н•ҳлҠ” state лҳҗлҠ” propsк°Җ мһҲлӢӨл©ҙ л°°м—ҙм—җ л„Јм–ҙмЈјлҠ” кІғмқҙ к·ңм№ҷмқҙлӢӨ. мҷңлғҗн•ҳл©ҙ н•ЁмҲҳмӢӨн–ү мӢң мөңмӢ  state лҳҗлҠ” propsлҘј м°ёмЎ°н•ҙм•ј н•ҳкё° л•Ңл¬ёмқҙлӢӨ.

useMemo

  • м—°мӮ°нҡҹмҲҳлҘј мӨ„м—¬ м„ұлҠҘмқ„ мөңм Ғнҷ”н•ҳкё° мң„н•ң Hook (stateліҖнҷ”лЎң re-renderingмӢңм—җ мғҲлЎң м—°мӮ°лҗҳм§Җ м•ҠлҸ„лЎқ)
  • useMemoмқҳ мІ« лІҲм§ё parameterм—җлҠ” м—°мӮ°мқ„ м •мқҳн•ң н•ЁмҲҳлҘј л„Јкі , л‘җ лІҲм§ё parameterм—җлҠ” л°”лқјліј нҠ№м • к°’мқ„ л°°м—ҙм—җ л„Јм–ҙмӨҖлӢӨ.
    • н•ҙлӢ№ к°’(нҠ№м •н•ң к°’)мқҙ л°”лҖҢл©ҙ н•ЁмҲҳк°Җ нҳём¶ңлҗҳкі , л§Ңмқј лӢӨлҘё мғҒнҷ©м—җ нҳём¶ңлҗҳм—ҲмңјлӮҳ н•ҙлӢ№ к°’мқҙ л°”лҖҢм§Җ м•Ҡм•ҳлӢӨл©ҙ мқҙм „м—җ м—°мӮ°н•ң к°’мқ„ мһ¬мӮ¬мҡ©н•ҳкІҢ лҗңлӢӨ.

useCallback

  • нҠ№м • н•ЁмҲҳлҘј мғҲлЎң л§Ңл“Өм§Җ м•Ҡкі , мһ¬мӮ¬мҡ© н•ҳкі  мӢ¶мқ„ л•Ң мӮ¬мҡ©н•ҳлҠ” Hook (stateліҖнҷ”лЎң re-renderingмӢңм—җ мғҲлЎң л§Ңл“Өм–ҙм§Җм§Җ м•ҠлҸ„лЎқ)
  • useMemoлҘј кё°л°ҳмңјлЎң л§Ңл“Өм–ҙ мЎҢмңјлӮҳ н•ЁмҲҳм—җ лҚ” мң мҡ©н•ҳкІҢ customлҗң кІғ
  • useCallbackл§ҢмңјлЎңлҠ” лҲҲм—җ лқ„лҠ” мөңм Ғнҷ”к°Җ м—ҶмңјлӮҳ component rendering мөңм Ғнҷ” мһ‘м—…мқҙ к°ҷмқҙ лҗҳл©ҙ м„ұлҠҘмқҙ мөңм Ғнҷ” лҗңлӢӨ.

н•ЁмҲҳ м•Ҳм—җм„ң мӮ¬мҡ©н•ҳлҠ” state лҳҗлҠ” propsк°Җ мһҲлӢӨл©ҙ л°ҳл“ңмӢң second parameterм—җ л°°м—ҙлЎң л„Јм–ҙмЈјм–ҙм•ј н•ңлӢӨ. (к·ёлһҳм•ј н•ЁмҲҳ лӮҙм—җм„ң state лҳҗлҠ” props м°ёмЎ°мӢң мөңмӢ мқҳ к°’мқ„ м°ёмЎ°н•ҳкё° л•Ңл¬ёмқҙлӢӨ.)
