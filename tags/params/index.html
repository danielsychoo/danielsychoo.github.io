useParams

paramsлҘј мқҙмҡ©н•ҙ routeлҘј м •н•  л•Ң

routeлҘј м„Өм •н•ҳлҠ” кіјм • мӨ‘ paramsлҘј мқҙмҡ©н•ҳкІҢ лҗ  кІҪмҡ° л°”лЎң paramsлҘј мқҙмҡ©н•ң routeлҘј м§Җм •н•ҳкё°ліҙлӢӨ м•һм—җ movie-detailsмҷҖ к°ҷмқҖ лӮҙмҡ©мқ„ н•Ёк»ҳ л„ЈлҠ”кІҢ мўӢмқҢ.

л§Ңмқј мқҙл ҮкІҢ н•ҳм§Җ м•ҠмқҖмұ„ paramsл§ҢмңјлЎң routeлҘј м„Өм •н•ҳл©ҙ м–ҙл–Ө к°’мқ„ л„Јм–ҙлҸ„ routeм—җ кұёл Ө 404 not found нҺҳмқҙм§ҖлҘј л¶„кё°н•  мҲҳ м—Ҷкё° л•Ңл¬ё.

мқҙлҘј кі л Өн•ҳм—¬ routeлҘј м„Өм •н•ңлӢӨл©ҙ м•„лһҳмҷҖ к°ҷмқҖ мҳҲмӢңк°Җ лҗ  кІғмһ„.

мң„ мІҳлҹј routeлҘј мһ‘м„ұн•ңлӢӨл©ҙ, userк°Җ м§Җм •н•ң routeмҷём—җ лӢӨлҘё кІҪлЎңлЎң м ‘мҶҚн–Ҳмқ„ л•Ң лӘЁл‘җ 404notFound нҺҳмқҙм§ҖлЎң мқҙлҸҷлҗ  кІғмһ„.

404 not found нҺҳмқҙм§Җ

мң„мқҳ мғҒнҷ©мқ„ л°ҳлҢҖлЎң н‘ңнҳ„н•ңлӢӨл©ҙ, мқҙлҠ” userк°Җ м •мқҳлҗҳм§Җ м•ҠмқҖ routeлҘј мҡ”мІӯн–Ҳмқ„ кІҪмҡ° 404 not found нҺҳмқҙм§ҖлЎң мқҙлҸҷлҗҳм–ҙм•ј н•ңлӢӨлҠ” кІғмқҙлӢӨ. мҰү, 404 нҺҳмқҙм§Җк°Җ кө¬м¶•лҗҳм–ҙ мһҲм–ҙм•ј н•ңлӢӨлҠ” кІғ.

м–ҙл–»кІҢліҙл©ҙ л„Ҳл¬ҙ лӢ№м—°н•ң м–ҳкё°мқёлҚ°, лӮҳлҠ” м§ҖкёҲк№Ңм§Җ мһҗлҸҷмңјлЎң 404 нҺҳмқҙм§ҖлЎң м—°кІ°лҗңлӢӨкі  мғқк°Ғн–Ҳм—ҲлӢӨ. л§Ңмқј 404 нҺҳмқҙм§ҖлҘј кө¬м¶•н•ҳм§Җ м•Ҡм•ҳлӢӨл©ҙ, userлҠ” м•„л¬ҙкІғлҸ„ м—ҶлҠ” кіөл°ұ нҺҳмқҙм§ҖлҘј л§ҲмЈјн•ҳкІҢ лҗңлӢӨ. мқҙлҠ” UXмёЎл©ҙм—җм„ңлҸ„ лӮҳмҒң мғҒнҷ©. (м—Ҷмңјл©ҙ м—ҶлӢӨкі лқјлҸ„ лқ„мӣҢмӨҳм•ј Userк°Җ мғҒнҷ©мқ„ мқём§Җн•Ё)

Apollo-client queryмқҳ keyмҷҖ type

Apollo-clientм—җм„ң queryлҘј лӮ лҰҙ л•Ң н•ҙлӢ№ queryм—җ ліҖмҲҳк°Җ мһҲмқ„ кІҪмҡ° ліҖмҲҳмқҳ keyлҘј кі л Өн•  кІғ.

мң„ мҪ”л“ңлҘј ліҙл©ҙ idлЎң keyлҘј м •н•ҙлҶ“кі  к·ё dataлҘј лӢҙм•„лҶ“мқҖ ліҖмҲҳмқҙлҰ„мқё ClickedIdлҘј л„Јкі мһҲмқҢ. мқҙ кІҪмҡ°
variables: { id: ClickedId }мҷҖ к°ҷмқҖ нҳ•мӢқмңјлЎң л„Јм–ҙм•ј н•Ё.

лҳҗн•ң н•ҙлӢ№ queryмқҳ value typeмқ„ IntлЎң м§Җм •н•ҙлҶ“м•ҳмңјлҜҖлЎң, нҶөмӢ мқ„ нҶөн•ҙ valueк°Җ StringмңјлЎң мҳЁлӢӨл©ҙ parsingн•ҙмӨҳм•ј н•ңлӢӨлҠ” м җмқ„ мһҠмңјл©ҙ м•ҲлҗЁ.

  • Stringмқ„ IntлЎң л°”кҝҖ л•Ң (parseInt(ліҖмҲҳ))
